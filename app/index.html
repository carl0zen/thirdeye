Third Eye Framework v 0.0.1

This framework uses the following technologies:

  • Backbone: MV* Pattern library
  • Marionette: For module integration
  • HTML5 Boilerplate
  • SASS
  • Compass
  • Sublime Text 3 snippets
  • Kendo-UI Controls
  • Modules:
    • Awesomes
    • Header
    • Content Manager
